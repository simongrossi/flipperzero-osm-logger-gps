Add track mode, dynamic presets, variants, note editor and Status view

Features:
- Mode trace GPX: new view + FuriTimer, writes track.gpx with <trkseg>
  per session (write-append-framed keeps file valid at all times)
- Dynamic presets: load /ext/apps_data/osm_logger/presets.txt on start
  (format: label;key;value[,variant1,...]), fallback to compiled defaults
- Preset variants: <- / -> cycle through alternative values for the
  same key (e.g. Cafe -> pub / bar / fast_food)
- Note editor: Up opens Flipper TextInput, Down clears; note is saved
  in JSONL / CSV for each point
- Fix acquired notification: ISR-posted flag consumed by the tick
  callback (vibrate + green LED on no-fix -> fix transition)
- Status GPS as dedicated view instead of a one-off action
- Total cumulative counter: counts points.jsonl lines at startup,
  shown as "t=N" on Quick Log
- About screen with author, GitHub link and license

Infra:
- .gitignore for dist/, .vscode/, .ufbt/, .DS_Store

Docs:
- New docs/ROADMAP.md with remaining features grouped by theme
- Updated README features list and Quick Log screen anatomy
- presets.txt.sample documents variants syntax
